# Kleindl_et_al_2026_Production_and_Enrichment_Shallow_Freshwater_Systems/shapefiles

Shapefiles included in analysis to create a map of Shark River Slough (SRS) and Taylor Slough (TS) in Everglades National Park (ENP) located in south Florida (FL), USA.

## FCE1294_FL_Shapefile and FCE1294_ENP_Shapefile

__FCE1274_FL_Shapefile__

A shape file of South Florida boundary lines used to create a map of data points sampled for this project. This shape file was retrieved from the State of Florida Boundary Line spatial data available as open source for download application for the Southwest Florida Water Management District.

__FCE1274_ENP_Shapefile__

A shape file of Everglades National Park obtained from the National Park Service (NPS, 2024) and used to create a map of data points sampled for this project. Reference: NPS - Land Resources Division. 2024. Administrative Boundaries of National Park System Units - National Geospatial Data Asset (NGDA) NPS National Parks Dataset. NPS - Land Resources Division

__Data citation:__

Kleindl, P.M., E. Gaiser, A. Wachnicka, J. Sah, and M. Ross. 2024. Macrophyte and microbial mat biomass co-variation along a hydrologic gradient and response to a removal experiment in temporary wetlands Everglades, FL, USA, February 2003 – November 2006 ver 3. Environmental Data Initiative. https://doi.org/10.6073/pasta/01ef607bc7090c3488e5bcb7475847e9 (Accessed 2026-04-07).

## FCE1294_SRS_Shapefile and FCE1294_TS_Shapefile

__FCE1294_SRS_Shapefile__

A shape file of the Shark River Slough boundary within the Everglades, which was created by Mike Rugge in 2001 by outlining the slough in a Landsat image.

__FCE1294_TS_Shapefile__

A shape file of the Taylor Slough boundary within the Everglades, which was created by Mike Rugge in 2001 by outlining the slough in a Landsat image.

Shape files were downloaded from the GitHub repository maps_FCE_sites_grayscale_base at https://github.com/FCE-LTER/maps_FCE_sites_grayscale_base.git
